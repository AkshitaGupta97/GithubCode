Create branch

To create a new branch, use the command: git checkout -b feature-branch

Push changes

After making changes, you can push them to the remote repository using:

git add . to stage changes, followed by git commit -m "Your message" to commit them.

To checkout on branch git checkout (branch name)

Finally, use git push origin feature-branch to push your changes to the remote repository.

For more information, check out the GitHub Documentation.

Happy coding!

Merge branch

Merge changes from the main branch into your feature branch.

To merge, use the command: git merge main

Pull changes

To pull the latest changes from the remote repository, use:

git pull origin main

This will update your local repository with the latest changes from the main branch.
